# 📂 Excel VBA CSV Consolidation Automation

## 📌 Project Overview

This project demonstrates an Excel VBA automation solution that consolidates multiple CSV files from a selected folder into a single master worksheet. The macro automatically imports data, removes duplicate headers, and combines all records into one dataset, significantly reducing manual effort and processing time.

---

## 🎯 Project Objective

The objective of this project is to:

* Automate the consolidation of multiple CSV files.
* Eliminate repetitive manual copy-and-paste tasks.
* Improve reporting efficiency.
* Create a single master dataset for analysis and dashboarding.

---

## 🛠️ Tools & Technologies

* Microsoft Excel
* VBA (Visual Basic for Applications)
* CSV Files
* Microsoft Office

---

## ⚙️ Automation Features

* Reads all CSV files from a specified folder.
* Automatically imports data from every file.
* Copies the header only once.
* Merges all remaining records into a single worksheet.
* Processes large datasets efficiently.
* Displays a completion message after consolidation.
